## 🥇 PRIORITÉ 1 — À contacter en premier

### 1. ADIE — Association pour le Droit à l'Initiative Économique
- **Ce qu'ils font** : Microcrédit jusqu'à 12 000 € + accompagnement gratuit
- **Pour qui** : Entrepreneurs exclus du crédit bancaire
- **Taux microcrédit** : ~7,45 % avec assurance chômage/décès incluse
- **Décaissement** : 8-15 jours après validation
- **Antenne 93** : Bobigny (via agence Paris Est ou Saint-Denis)
- **Contact** : 0 969 328 110 (gratuit) — https://www.adie.org
- **Docs à préparer** : CIS, attestation SIRENE, doc Johny prévisionnel, RIB

### 2. BGE Parif — Boutique de Gestion Paris Île-de-France
- **Ce qu'ils font** : Accompagnement business plan + prévisionnel (subventionné 100-300 €)
- **Adresse 93** : Antenne Saint-Denis + Bobigny
- **Site** : https://www.bge-parif.com
- **Force** : Consultant dédié qui construit le prévisionnel 36 mois avec toi
- **Souvent gratuit si Pôle Emploi ou RSA**

### 3. Bpifrance Création — Outil gratuit en ligne
- **Ce qu'ils font** : Générateur prévisionnel automatique + ratios sectoriels
- **Site** : https://bpifrance-creation.fr
- **Coût** : 100 % GRATUIT
- **Avantage** : Standard bancaire, accepté par SG
- **Limite** : Générique, moins personnalisé qu'un vrai humain

- **Force** : Peut te donner une **garantie sur les 2 000 € SG** (baisse le risque perçu par la SG)

- **Zone** : Départements 93 + 95
- **Site** : https://initiative-nord-idf.fr
- **Avantage** : Prêt à 0 %, remboursement 3-5 ans
